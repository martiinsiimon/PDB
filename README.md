PDB - projekt pdb13
===================

Navod k instalaci:
 * V korenovem adresari chybi slozka 'lib', ve ktere maji byt nasledujici knihovny:
   * ojdbc6.jar
   * ordim.jar
   * sdoapi.jar
   * Poznamka: Vsechny knihovny jsou dostupne z adresy https://www.fit.vutbr.cz/homes/rychly/private/pdb/lib/
   
Prelozeni projektu:
 * ant compile
 
Spusteni projektu:
 * ant run
